# 3-D-Elastic-Time-Reverse-Imaging-of-a-Linear-Void-Anomaly-Example
2D synthetic examples from the paper "3-D Elastic Time-Reverse Imaging of a Linear Void Anomaly"

To view, zip files in the SConstruct folder unpack to provide examples that run using the open source software Madagascar (https://ahay.org/wiki/Main_Page). All SConstructs have been set up for the option of running manually or parallelized between nodes.

Supplimental files needed to build models and run the E-TRI migration can be found in the Supplimental folder. These should be downloaded and set up prior running, and the path changed within the SConstruct. These functions requre a GPU computer.
